Адаптивная HTML вёрстка

Осуществляю кроссбраузерную адаптивную вёрстку сайтов по методологии БЭМ (flex, Grid, animation, Pixel Perfect, SVG) по макетам Figma и Photoshop.

Сторонние библиотеки

Уверенно использую скрипты и подключаю сторнние библиотеки JavaScript, JQuery, Bootstrap.

Применяю в работе препроцессоры SASS/LESS. При необходимости, использую таск менеджер Gulp для автоматизации вёрстки.

Командная работа

Способен работать в команде, используя систему контроля версий Git.
