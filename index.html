Мои навыки

Адаптивная HTML вёрстка

Осуществляю кроссбраузерную адаптивную вёрстку сайтов по методологии БЭМ (flex, Grid, animation, Pixel Perfect, SVG) по макетам Figma и Photoshop.

Сторонние библиотеки

Уверенно использую скрипты и подключаю сторнние библиотеки JavaScript, JQuery, Bootstrap.

Применяю в работе препроцессоры SASS/LESS. При необходимости, использую таск менеджер Gulp для автоматизации вёрстки.

Командная работа

Детали проекта:

Свёрстана только главная страница сайта. Сайт адаптивный. Использовался препроцессор SASS. Модальные окна на JQuery. Подключен слайдер Swiper и реализована валидация полей форм на JQuery. Отправка данных форм на библиотеке PHPMailer. Использована анимация на библиотеке AOS и элементы CSS анимации . Подключена Google карта. Сайт свёрстан в рамках обучения на курсе “Веб-старт” учебной платфомы “GLO Academy”.
